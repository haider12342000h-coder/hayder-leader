# منصة الاستشارات الطبية

واجهة React حديثة باللغة العربية مع تصميم RTL و`Tailwind CSS` و`React Router`.

## التشغيل

استخدم:

```bash
npm install
npm run dev
```

إذا كان PowerShell يمنع `npm`, يمكن تشغيل:

```bash
npm.cmd install
npm.cmd run dev
```

## هيكل المشروع

```text
src/
  components/
    common/
    navigation/
  data/
  layouts/
  pages/
  routes/
  styles/
```
